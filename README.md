JavaScriptCore
==============

Unofficial mirror of the JavaScript of WebKit  http://www.webkit.org/

## Build

Prerequisites(Ubuntu 12.04):

    * glib 2.3.8 or lader
    * libsoup 2.4
    * gtk+-3.0
    * cmake
    * libgstreamer

###steps:
     cd JavaScriptCore
     ./build-jsc
